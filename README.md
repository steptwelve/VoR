# VoR - Daily Meditation Scraper & Poster

An automated tool that scrapes daily meditations and posts them to social media platforms (BlueSky and X/Twitter) with generated images.

## Overview

This project automatically:
- Scrapes daily meditation content
- Generates an accompanying image
- Posts to BlueSky (@ocisaa.org)
- Posts to X/Twitter
- Runs via GitHub Actions workflow

## Features

- **Automated Daily Posts**: Scheduled via GitHub Actions
- **Multi-Platform**: Posts to BlueSky and X simultaneously
- **Visual Content**: Generates custom images for each meditation
- **Zero Maintenance**: Runs automatically without manual intervention

## Usage

This project is designed to run automatically via GitHub Actions. The workflow is scheduled to run daily and post the meditation content to configured social media accounts.

## Configuration

Requires the following GitHub Secrets:
- BlueSky API credentials
- X/Twitter API credentials
